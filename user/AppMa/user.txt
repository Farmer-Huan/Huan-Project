# 1. 용량
#  - Get 방식은 URL 뒤에 데이터가 쿼리스트링으로 첨부 되기때문에 길이의 제한이 있다.
#  - 게시판의 내용에 글을 쓸 때 데이터가 많아서 Get 방식으로는 한계가 있다.

#  - Post 방식은 인코딩 필요 없다.

# 3. 보안상 문제 
#  - Get 방식으로 ID & Password를 넘길 때에 URL이 보이기 때문에 보안상 문제점이 있다.
